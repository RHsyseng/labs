Description

These are the folders where the manifests to deploy operators (AI, Hive, ACM) and resources (BMH…) are located. Just only with these folders should be enough to deploy an operator on top of an OCP cluster deployed before.

Inside this docs tree should be one doc for each kind of deployment /deploy-* explaining the way to do manually like a reference doc.

For instance, deploy-AssistedInstaller-operator, could contain:

  • Manifests to deploy the operator
  • Sometimes, the manifests should contain PLACEHOLDER just to be replaced and automated by the workflow (using GitHub Actions inputs to set the version, or w/e you need)

You could use these docs to follow the deployment process by hand instead of doing with the workflow pipeline. Just replace with your variable and launch it manually.
